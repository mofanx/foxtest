## S2-1: M1 AI 探索循环实战验证

### 目标
验证 AI + dp-cli 技能的协作，完整走通：
```
自然语言用例 → dp-cli 探索 → IR 产出 → codegen → 回放验证
```

### 验证步骤
1. 确认 dp-cli skill 已安装
2. 使用 SKILL.md 工��流处理一个真实用例
3. 验证产出的 IR 能通过 validate_ir.py
4. 验证生成的脚本回放通过

### 成功标准
- AI 能正确理解用例并指导 dp-cli 探索
- 产出的 IR 合法且可回放
- 完整链路无人工干预完成
